void LiftCube::begin ( ) [private]

Prepares the LiftCube object to work according to the values defined in the configuration. The preparations done in here can not be done from within the constructor because of dependencies such as Serial communication. This method needs to be called before this class can be fully used. In this class it is done when getting an instance of the class the first time.

void LiftCube::doJob ( )

Checks if it's already time to lift up the hoist again after liftCube() was called. This method has to be called in regular intervals (as short as possible).

void LiftCube::liftCube ( )

Moves the hoist to the bottom position (according to the configuration), waits for the time defined in Configuration::liftCubeDownUpDuration and then moves the hoist to the top position again.
Make sure the doJob method is called in regular intervals.

See also:

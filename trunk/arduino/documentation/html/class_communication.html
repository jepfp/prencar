void Communication::send ( byte  messageId,
long const  params[],
byte  paramSize 
)

Sends the message code over the configured serial interface.
The message will be filtered according to the configured filter level in the configuration.

The message will be sent in the following format:
202-4:12,323,4399,2292\n
[messageId]-[sizeOfParameters]:[parameter 1],[parameter 2],[parameter 3],[parameter n]\n

Parameters:
messageIdId of the message that will be sent.
params[]Parameters that will be sent with this message.
paramSizeSize of params[].

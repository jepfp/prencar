int Configuration::lineFollowActivateFrontSensorOffset

Milliseconds (after a call to startIt() of LineFollow) after which the front line sensors shall be activated.

This offset is helpful at the beginning of the parours in order to be able to start the parcours behind the start line and after driving a curve if at the end of the curve one of the front line sensors still is on top of the 90°-line. If set to 0 the front line sensors are always on.

int Configuration::lineFollowWhiteThresholdLineSensors

Threshold value from black to white.

If the sensor measures a value below that threshold value than the software considers that sensors physical position to be on "white" ground. This treshold value is calculated during the calibration of the sensors.
The default value 156 is based on first measurements and set as long as no calibration is done.

See also:
LineFollow::calibrateSensors()

void MoveBackToLine::doJob ( ) [virtual]

Does the job for which this class was written.
This method has to be called in regular intervals (as short as possible).
The left or right sensor (depending on the value _reachingLineFromLeft) is checked to see if the line has already been reached. If so the line centering process is done.

boolean MoveBackToLine::_reachingLineFromLeft [private]

reachingLineFromLeft = true means that the car is reaching the line from the left hand side so the car has to check the left line sensor. reachingLineFromLeft = false means that the car is reaching the line from the right hand side so the car has to check the right line sensor.

boolean MoveBackToLine::errorNoCubeApproachInformation

Set to true if no cube approach information has been found.

The 'move back to line' process can not be started if there is no track back (cube approach recordings which have informations about how the line has been left are empty). In that case, the error flag is set to true.

boolean MoveBackToLine::lineHasBeenReached

This variable is set to true as soon as the line has been reached but the car is not yet centered.

With other words this means that the left (or right) line sensor has detected white ground so we can continue with the normal line following.
